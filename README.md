## Crear un proyecto

```bash
go mod init github.com/celsiainternet/taller
```

## Ejecutar el proyecto

```bash
go run ./main.go
```

## Servicio API REST

```bash
go get github.com/go-chi/chi/v5
go run ./api/main.go
```

## Clone repository

```bash
git clone https://github.com/CelsiaInternet/taller.go.git go
git set develop
git new <name>

```
